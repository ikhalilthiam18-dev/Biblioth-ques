Authentification JWT

Sécurisez vos requêtes avec un token d'accès Bearer

🔐 Obtenir un token

Envoyez vos identifiants à /api/auth/token/ pour recevoir un access token (60 min) et un refresh token (24h).

Incluez le token dans chaque requête protégée dans le header Authorization.


⚡ Permissions

  • ✅ Lecture — Accès public (livres, auteurs)
  • 🔐 Création — Utilisateur authentifié
  • ✏️ Modification — Propriétaire ou admin
  • 🗑️ Suppression — Propriétaire ou admin
Obtenir un token
# POST /api/auth/token/ { "username": "votre_username", "password": "votre_mot_de_passe" } # Réponse { "access": "eyJ0eXAiOiJKV1Qi...", "refresh": "eyJ0eXAiOiJKV1Qi..." }
Utiliser le token
# Header de requête Authorization: Bearer eyJ0eXAiOiJKV1Qi... # Exemple POST /api/livres/ curl -X POST /api/livres/ \ -H "Authorization: Bearer {token}" \ -H "Content-Type: application/json" \ -d '{"titre": "...", "isbn": "...", "auteur": 1}'
